Project Description

Le projet REST Countries API est une application web qui affiche des informations détaillées sur tous les pays du monde, y compris leur population, leur capitale et d’autres données clés. Ce projet m’a permis d’explorer la récupération et l’affichage dynamique de données depuis une API externe, tout en offrant une interface intuitive aux utilisateurs. L’expérience a été globalement fluide et agréable à réaliser, car la gestion des requêtes API et l’affichage des données ne posaient pas de difficultés particulières. Ce projet illustre ma capacité à manipuler des API et à concevoir des interfaces interactives et informatives.

Solution

Le principal défi de ce projet a été la mise en place d’un système de filtrage efficace des pays selon différents critères (nom, région, population, etc.). J’ai rencontré quelques difficultés pour optimiser ce filtrage, notamment au niveau des performances et de la gestion des entrées utilisateur. Après avoir cherché des solutions et échangé avec d’autres développeurs ayant rencontré les mêmes problématiques, j’ai réussi à implémenter un filtre performant et réactif. Ce projet m’a conforté dans l’utilisation des API REST et m’a appris à surmonter les obstacles techniques en m’appuyant sur la communauté de développeurs.
